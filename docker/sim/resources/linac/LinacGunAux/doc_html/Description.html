LinacGunAux Class

Revision: Release_1_0 - Author: bourtemb - CVS repository: ESRF
This  class manages the gun auxiliary of the Linac.


A tungsten filament is heated in the gun to create electrons.
This class allows to manage the heating of the filament ie to change the Heating current, 
to known the heating remaining time and to switch on or off the filament heating.
